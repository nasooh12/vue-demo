## 변경사항 

Vue2 예제 코드를 Vue3 Composition API 중심으로 리팩터링.
<script setup>, reactive API, defineProps/Emits, TypeScript 적용.
동일 UI/동작 유지하며 최신 Vue 스타일로 구조 정리.
